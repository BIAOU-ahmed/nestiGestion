Convention de Nommage :

1) Diagramme de classe :

- Nous avons utilisé le "CamelCase" dans le diagramme de classe.
- Langage en anglais.

2) MCD :

- Nous avons utilisé le "lowerCase" dans le MCD.
- Le nom des entités est en UpperCase.
- Le nom des associations est en minuscules.

3) UseCase :

- Nous avons utilisé un nommage literale
- Langage en anglais
